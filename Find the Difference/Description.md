# [389. Find the Difference](https://leetcode.com/problems/find-the-difference/)

## 🟢 Easy

---

### **Problem Statement**

You are given two strings `s` and `t`. String `t` is generated by randomly shuffling string `s` and then adding one more letter at a random position.

**Goal:** Return the letter that was added to `t`.

---

### **Examples**

**Example 1:**
> **Input:** `s = "abcd", t = "abcde"`  
> **Output:** `"e"`  
> **Explanation:** 'e' is the letter that was added.

**Example 2:**
> **Input:** `s = "", t = "y"`  
> **Output:** `"y"`

---

### **Constraints**

- $0 \le s.length \le 1000$
- $t.length = s.length + 1$
- `s` and `t` consist of lowercase English letters.

---

### **Complexity Analysis**

* **Time Complexity:** $O(n)$  
  We iterate through both strings once.
* **Space Complexity:** $O(1)$  
  Using either Bit Manipulation or the Sum approach requires only a single variable to store the result.

---

### **Approach 1: Bit Manipulation (XOR)**



This is the most "elegant" solution. Recall that $x \oplus x = 0$ and $x \oplus 0 = x$. If we XOR every character in both $s$ and $t$, the characters that are identical in both strings will cancel each other out, leaving only the extra character.

#### **The Strategy:**
1. Initialize a variable `res = 0`.
2. XOR every character in `s` into `res`.
3. XOR every character in `t` into `res`.
4. The remaining value in `res` is the ASCII value of the added letter.

---

### **Approach 2: Character Sum (ASCII Math)**

Since $t$ is just $s$ plus one extra character, the sum of ASCII values in $t$ minus the sum of ASCII values in $s$ will equal the ASCII value of the added character.

#### **The Strategy:**
1. Sum all character codes in string `t`.
2. Subtract all character codes in string `s`.
3. Convert the resulting difference back into a character.

---

### **Approach 3: Hash Map / Frequency Array**

Similar to **Ransom Note**, you can count the frequencies of characters in $s$ and decrement them using $t$. The character that ends up with a negative count (or is missing from $s$) is the added letter.
