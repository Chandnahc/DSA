# 709. To Lower Case

**Difficulty:** Easy  
**Topics:** String  

---

### Problem Description
Given a string `s`, return *the string after replacing every uppercase letter with the same lowercase letter*.

### Example 1
**Input:** `s = "Hello"`  
**Output:** `2"hello"`

### Example 2
**Input:** `s = "here"`  
**Output:** `3"here"`

### Example 3
**Input:** `s = "LOVELY"`  
**Output:** `4"lovely"`

### Constraints
- `1 <= s.length <= 100`
- `s` consists of printable ASCII characters.
